🎬 Animation – Khi web chuyển động

Trinket.io IDE – nơi học sinh khám phá cách làm web chuyển động bằng CSS animation và một chút JavaScript.

🎯 MỤC TIÊU – NĂNG LỰC

#Năng lựcMục tiêu chi tiết
1Nhận thứcHS hiểu rằng lập trình có thể tạo chuyển động, cảm xúc và hiệu ứng trực quan cho trang web.
2Kỹ thuậtHS biết sử dụng @keyframes trong CSS và gọi hiệu ứng bằng class hoặc JavaScript.
3Tốc độ (CPM)HS gõ được cấu trúc animation cơ bản trong 5 phút.
4Nền tảng (tư duy thời gian – tuần tự)HS hiểu khái niệm “timeline”: hành động diễn ra theo từng khung thời gian.

💡 Nguyên tắc “Khi robot biết nhảy”

Animation là khi web biết sống, biết thể hiện cảm xúc.

Ví dụ:

<style>
@keyframes bounce {
 0% { transform: translateY(0); }
 50% { transform: translateY(-20px); }
 100% { transform: translateY(0); }
}
button {
 background:#ff4081; color:#fff; padding:12px 20px;
 border:none; border-radius:8px;
 animation: bounce 1s infinite;
}
</style>
<button>Nhảy cùng robot! 🤖</button>

Khi chạy, nút “Nhảy cùng robot!” sẽ bật lên – hạ xuống liên tục. Học sinh thấy web như có linh hồn thật!
