🧠 If/Else – Ra quyết định

Trinket.io IDE – nơi học sinh học cách cho máy tính “suy nghĩ có điều kiện”: nếu đúng thì làm A, nếu sai thì làm B.

🎯 MỤC TIÊU – NĂNG LỰC

#Năng lựcMục tiêu chi tiết
1Nhận thứcHS hiểu rằng máy tính cũng có thể “ra quyết định” dựa trên điều kiện.
2Kỹ thuậtHS biết cú pháp if (...) { ... } else { ... } và hiểu cách hoạt động.
3Tốc độ (CPM)HS gõ được cấu trúc if/else hoàn chỉnh trong 3 phút.
4Nền tảng (tư duy logic)HS biết phân tích tình huống: nếu... thì..., ngược lại thì...

💡 Nguyên tắc “Khi robot biết suy nghĩ”

If/Else là cặp não trái – não phải của lập trình viên.

Ví dụ:

<script>
  let mood = prompt("Bạn đang vui hay buồn?");
  if (mood == "vui") {
    alert("🤖 Robot cũng vui theo bạn!");
  } else {
    alert("🤖 Robot gửi bạn một cái ôm! ❤️");
  }
</script>

Khi chạy, nếu học sinh nhập “vui”, robot chúc mừng; nếu nhập khác, robot an ủi – giống như bạn thân biết cảm xúc của mình vậy!

📘 CHÚ THÍCH THUẬT NGỮ CHUYÊN MÔN

Thuật ngữNghĩaGợi ý cách nói cho trẻ
ifTừ khóa kiểm tra điều kiện.“Nếu đúng thì làm…”
elseTừ khóa thực hiện khi điều kiện sai.“Nếu không thì làm khác.”
conditionĐiều kiện kiểm tra (so sánh, giá trị).“Câu hỏi của robot.”
comparisonPhép so sánh như ==, >, <.“Robot kiểm tra xem đúng không.”
booleanGiá trị đúng/sai (true/false).“Có hoặc không.”
prompt()Hàm để nhập dữ liệu.“Robot hỏi bạn điều gì đó.”
alert()Hàm hiển thị thông báo.“Robot trả lời bạn.”
else ifĐiều kiện trung gian (nhiều khả năng).“Nếu không phải vậy, thì thử điều khác.”
logicTư duy điều kiện và kết quả.“Suy nghĩ như robot.”
