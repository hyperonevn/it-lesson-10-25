GIÁO ÁN LẬP TRÌNH TRẺ EM – TRINKET.IO Bài 12 10/2025

🧩 Ghép nối HTML + CSS + JS – Làm web hoàn chỉnh

Trinket.io IDE – nơi học sinh nhìn thấy bức tranh tổng thể: HTML (khung), CSS (màu sắc), và JS (chuyển động) cùng hòa quyện thành một website hoàn chỉnh.

🎯 MỤC TIÊU – NĂNG LỰC

#Năng lựcMục tiêu chi tiết
1Nhận thứcHS hiểu rằng một website hoàn chỉnh là sự kết hợp của 3 “người bạn”: HTML – CSS – JavaScript.
2Kỹ thuậtHS biết liên kết file CSS và JS vào HTML bằng <link> và <script>.
3Tốc độ (CPM)HS gõ được cấu trúc dự án 3 file (index.html, style.css, script.js) trong 5 phút.
4Nền tảng (tư duy tích hợp)HS biết tổ chức, chia nhỏ và kết hợp nhiều phần lại thành hệ thống hoàn chỉnh.

💡 Nguyên tắc “Ba người bạn hợp sức xây web”

HTML – dựng khung, CSS – sơn màu, JavaScript – thổi hồn.

Ví dụ:

<!-- index.html -->
<!DOCTYPE html>
<html>
 <head>
  <title>Trang web đầu tiên của con</title>
  <link rel="stylesheet" href="style.css">
 </head>
 <body>
  <h1>Chào mừng đến với thế giới lập trình!</h1>
  <button onclick="changeColor()">Đổi màu nền</button>
  <script src="script.js"></script>
 </body>
</html>

<!-- style.css -->
body { font-family: 'Comic Sans MS'; text-align:center; background-color:#fef9c3; }
h1 { color:#ff4081; }

<!-- script.js -->
function changeColor() {
 document.body.style.backgroundColor = '#a5f3fc';
}

Khi chạy, trang có giao diện đẹp, nút bấm đổi màu – trẻ thấy sự kết nối rõ ràng giữa ba phần.
