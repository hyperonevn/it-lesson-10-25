🤖 JavaScript cơ bản – Khi web biết phản ứng

Trinket.io IDE – công cụ giúp học sinh chạy thử code JavaScript trực tiếp trong trình duyệt và thấy phản ứng ngay lập tức.

🎯 MỤC TIÊU – NĂNG LỰC

#Năng lựcMục tiêu chi tiết
1Nhận thứcHS hiểu rằng JavaScript là ngôn ngữ giúp web “chuyển động” và “phản ứng” với người dùng.
2Kỹ thuậtHS biết tạo hàm và gắn sự kiện (onclick) để thực hiện hành động khi bấm nút.
3Tốc độ (CPM)HS gõ được đoạn code JavaScript đơn giản (hiện hộp thoại alert) trong 3 phút.
4Nền tảng (tư duy nguyên nhân – kết quả)HS hiểu “sự kiện” là nguyên nhân, “hành động của web” là kết quả.

💡 Nguyên tắc “Khi web biết nghe lời”

HTML là người nói, CSS là người trang trí, JavaScript là người hành động.

Ví dụ:

<button onclick="sayHello()">Nhấn vào đây</button>
<script>
  function sayHello() {
    alert("Xin chào! Mình là robot JS 👋");
  }
</script>

Khi học sinh bấm nút, trang web hiển thị lời chào – giống như robot thật biết đáp lại!
