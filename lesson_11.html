🎨 JS đổi màu nền – Thí nghiệm vui

Trinket.io IDE – nơi học sinh thực hiện thí nghiệm nhỏ: dùng JavaScript để thay đổi màu nền trang web theo ý muốn.

🎯 MỤC TIÊU – NĂNG LỰC

#Năng lựcMục tiêu chi tiết
1Nhận thứcHS hiểu JavaScript có thể thay đổi trực tiếp giao diện web (màu sắc, hình dạng).
2Kỹ thuậtHS biết truy cập phần tử HTML bằng document.body và đổi thuộc tính bằng style.
3Tốc độ (CPM)HS gõ đoạn code đổi màu nền trong 3 phút và hiểu ý nghĩa từng dòng.
4Nền tảng (tư duy tác động – phản hồi)HS nhận biết mối liên hệ “khi hành động → giao diện thay đổi”.

💡 Nguyên tắc “Lập trình là phép màu thay đổi thế giới”

Với mỗi cú click, ta thay đổi cả không gian web.

Ví dụ:

<button onclick="changeColor()">Đổi màu nền! 🎨</button>
<script>
  function changeColor() {
    const colors = ["#fef3c7", "#d1fae5", "#e0e7ff", "#fde68a", "#fecaca"];
    const random = Math.floor(Math.random() * colors.length);
    document.body.style.backgroundColor = colors[random];
  }
</script>

Khi nhấn nút, màu nền thay đổi ngẫu nhiên – như một “bầu trời” luôn đổi sắc!
