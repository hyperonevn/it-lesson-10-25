🎨 CSS – Làm đẹp cho web

Trinket.io IDE – nơi học sinh tiếp tục phát triển “ngôi nhà web” đã xây bằng HTML, thêm màu sắc, bố cục và cá tính riêng.

🎯 MỤC TIÊU – NĂNG LỰC

#Năng lựcMục tiêu chi tiết
1Nhận thứcHS hiểu rằng CSS là ngôn ngữ giúp “trang trí và phối màu” cho trang web HTML.
2Kỹ thuậtHS biết dùng cú pháp CSS cơ bản: selector { property: value; } để đổi màu, phông chữ, căn lề.
3Tốc độ (CPM)HS gõ được 3 quy tắc CSS đúng cú pháp trong 3 phút.
4Nền tảng (thẩm mỹ – logic)HS biết chọn màu, font và bố cục hài hòa; hiểu “tách phần nội dung và phần trang trí”.

💡 Nguyên tắc “Làm đẹp cho web”

HTML là khung, CSS là sơn, nội thất và ánh sáng.

Ví dụ:

<style>
body { background-color: #fef9c3; font-family: 'Comic Sans MS'; }
h1 { color: #ff4081; text-align: center; }
p { color: #374151; font-size: 18px; }
</style>

Khi chạy, trang web trở nên rực rỡ hơn, có màu nền nhẹ và chữ xinh xắn – giống như một căn phòng được sơn và trang trí cẩn thận.
